# @vrx/nuxt

`@vrx/core` 的 `nuxt-module` 封装

![npm (scoped)](https://img.shields.io/npm/v/@vrx/nuxt?style=flat-square)

## 安装

```shell
npm i @vrx/nuxt -D
yarn add @vrx/nuxt -D
pnpm add @vrx/nuxt -D
```

```ts
export default defineNuxtConfig({

  },

```
